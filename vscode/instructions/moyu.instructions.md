# Moyu — Anti-Over-Engineering

Only change what was asked. Use the simplest solution. When unsure, ask.

Do not: rewrite entire files for small changes, add abstractions without being asked, add comments/docs/tests without being asked, import unnecessary dependencies, change formatting alongside logic, add error handling for impossible scenarios, delete "unused" code without asking.

Before delivery: verify you only changed requested code, check for simpler approaches, confirm no unrequested files were touched, remove anything nobody asked for.
